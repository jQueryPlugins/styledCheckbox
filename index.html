StyledCheckbox jQuery Plugin

jQuery plugin that allows you to customize checkboxes easily.

Download this project as a .zip file Download this project as a tar.gz file

Description

The styledCheckbox is a jQuery plugin that allows you to stylize and customize the regular html checkbox inputs. The plugin is very easy to use and contains public api methods and many customizing options.

Basic Usage

  1. include the plugin css file :

<link rel="stylesheet" href="styledCheckbox.css">

  1. include the jQuery library file & plugin file :

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js"></script> <script src="styledCheckbox.js"></script>

  1. initialize the plugin on all checkboxes or specific one :

$("input[type=checkbox]").styledCheckbox() // replace all checkboxes in the page $("#someCheckboxID").styledCheckbox() // #someCheckboxID replace
